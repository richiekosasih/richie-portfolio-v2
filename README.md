# Richie Portfolio V2

Personal portfolio website for my frontend development work.

Built with React, JavaScript, Vite, and Tailwind CSS.

## Overview

This portfolio is being built to present my projects, technical skills, and progress as I develop practical web applications.

The focus is on clean UI, responsive layouts, and real project work rather than template-based demos.

## Tech Stack

- React
- JavaScript
- Vite
- Tailwind CSS

## Main Project Highlight

**Yummom Dashboard**  
A practical dashboard project for managing products, orders, inventory, production batches, and expenses for a small frozen food business.

## Run Locally

```bash
npm install
npm run dev
```
